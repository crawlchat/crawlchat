date: 2026-03-27
type: changelog
tags: focus
---

Now you can show an interactive _API Playground_ widget that is embedded in the chatbot answers where the information is prefilled by the AI and the user can enter extra information and try out the APIs without leaving the chat widget.

It supports headers, body payload, query parameters, URL parameters as of now. The _AI_ uses the documentation (knowledge base) to decide the schema and URLs.

You can enable this from [settings page](https://crawlchat.app/settings#api-playground)
