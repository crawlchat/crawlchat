Now you can view and manage your collections, questions, and most of the things you can view on dashboard over an MCP server. This lets you integrate your CrawlChat collection information with your AI applications.

![MCP server for collections](/changelog-images/mcp-server-collections.png)

Read the full docs at [MCP API docs](https://docs.crawlchat.app/api/mcp).
